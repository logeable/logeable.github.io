Maximum Depth of Binary Tree

Given a binary tree, find its maximum depth. The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node. https://oj.leetcode.com/problems/maximum-depth-of-binary-tree/ class Solution { public: int maxDepth(TreeNode *root) { if(!root) { return 0; } int left=maxDepth(root->left); int right=maxDepth(root->right); return left<right?(right+1):(left+1); } };

Maximum Subarray

Find the contiguous subarray within an array (containing at least one number) which has the largest sum. For example, given the array [−2,1,−3,4,−1,2,1,−5,4], the contiguous subarray [4,−1,2,1] has the largest sum = 6.
